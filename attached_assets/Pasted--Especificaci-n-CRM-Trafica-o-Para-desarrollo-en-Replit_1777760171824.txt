# Especificación CRM Traficaño
## Para desarrollo en Replit

---

## 1. GESTIÓN DE CONTACTOS Y EMPRESAS

### Contactos
- [ ] Ficha de contacto: nombre, apellido, email, teléfono, celular, cargo, LinkedIn
- [ ] Foto de perfil (upload o LinkedIn)
- [ ] Múltiples emails y teléfonos por contacto
- [ ] Etiquetas/tags personalizables
- [ ] Campos custom ilimitados
- [ ] Historial completo de interacciones (timeline)
- [ ] Vinculación a múltiples empresas
- [ ] Score de contacto (lead scoring manual o automático)
- [ ] Estado: activo, inactivo, do-not-contact
- [ ] Fuente de origen (feria, web, referido, etc.)
- [ ] Notas y comentarios con @menciones
- [ ] Archivos adjuntos por contacto

### Empresas/Cuentas
- [ ] Ficha de empresa: razón social, CUIT, industria, tamaño, facturación estimada
- [ ] Dirección con geolocalización
- [ ] Múltiples sucursales/ubicaciones
- [ ] Jerarquía de empresas (matriz → subsidiarias)
- [ ] Contactos vinculados con rol (decisor, técnico, compras, etc.)
- [ ] Segmentación: cliente, prospect, ex-cliente, proveedor, competidor
- [ ] Clasificación: A, B, C (Pareto 80/20)
- [ ] Campos custom por industria (ej: consumo anual de caños en toneladas)
- [ ] Historial de compras
- [ ] Crédito y condiciones de pago
- [ ] Documentos vinculados (contratos, certificados)

## 5. AUTOMATIZACIONES (WORKFLOWS)

### Motor de automatización
- [ ] Trigger → Condición → Acción
- [ ] Triggers: nuevo contacto, deal cambia etapa, tarea vencida, email abierto, etc.
- [ ] Condiciones: campo = valor, fecha, vendedor, etc.
- [ ] Acciones: crear tarea, enviar email, notificar, cambiar campo, asignar owner
- [ ] Delays (esperar X días)
- [ ] Bifurcaciones (if/else)
- [ ] Builder visual drag-and-drop
- [ ] Templates de automatización predefinidos
- [ ] Log de ejecución de automatizaciones

### Ejemplos para Traficaño
- [ ] Cotización sin respuesta 5 días → email + tarea
- [ ] Deal >100k USD → notificar a Mariano
- [ ] Nuevo lead de feria → asignar a vendedor de turno
- [ ] Cliente sin actividad 90 días → alerta de reactivación
